----------------------
Which code is executed when something is clicked?
the function "gameSetupRound" is called. It checks if there's a square clicked and
places a ship ("placeShip(i)") if there's ships left and if it's placable. When there 
are no ships left, it creates the enemy sea ("createEnemySea()") and changes the onclick
listeners:

function gameSetupRound(clickEvent) {
    mouseX = clickEvent.offsetX
    mouseY = clickEvent.offsetY

    /* Check to sea whether or not the click was on one of the squares */
    for (var i = 0; i < mySquares.length; i++) {
        if (mySquares[i].isClicked(mouseX, mouseY)) {
            placeShip(i)
            if(currentShipIndex == shipSizes.length){
                createEnemySea()
                /* Change the onclick listeners to start the game. */
                document.getElementById("enemySea").onclick = playRound
                document.getElementById("mySea").onclick = ""
            }
        }
    }
}

What does the createMySea() function do?

function createMySea(){
    for (var rowNumber = 0; rowNumber < numRows; rowNumber++) {

What is the difference between a square on the canvas and a square 
in the mySquares array?
A square on the canvas is drawn and doesn't hold any variables/ values. The squares
in the mySquares array hold values and even methods.

What does the createFriendlyGridSquare() function do?
It creates a object called square and returns it:

function createFriendlyGridSquare(rowNumber, columnNumber){
    var square = {
        rowNr: rowNumber,
        columnNr: columnNumber,
        state: "water",
        getColor: mySeaColors,
        draw: drawSquare,
        changeState: changeSquareStateAndRedraw,
        isClicked: coordinatesOverlap
    }
    return square
}

What does the changeSquareStateAndRedraw() function do? 
Try to get it to do something by calling it (with the proper arguments) 
in the console.

(the possible states are found in mySeaColors() and enemySeaColors()) and redraws it:

function changeSquareStateAndRedraw(newState,context) {
    this.state = newState

What does the createEnemySea() function do? 
How does it differ from the createMySea() function?
It uses a for loop to go trough all the squares in the array. It uses 
insertEnemyGameMethods(index) to "add" the methods to the array and it  then draws the
"updated" square on the enemyContext

function createEnemySea(){
    for (rowNumber = 0; rowNumber < numRows; rowNumber++) {
        for (columnNumber = 0; columnNumber < numRows; columnNumber++) {
            var squareIndex = rowNumber * numColumns + columnNumber
            insertEnemyGameMethods(squareIndex)
            enemySquares[squareIndex].draw(enemyContext)
        }
    }
}
----------------------
What does currentShipIndex represent?
It is a global variable and represents the ammount of ships that have been placed
on the grid by the player. Used to "begin" the game loop.
